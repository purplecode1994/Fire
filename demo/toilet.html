AFC213(H)・AFC214(H)

智慧馬桶｜遙控器設定與配對

適用型號：AFC213(H)、AFC214(H)
1-1

設定時間

  1. 遙控器：長按 停止 → 螢幕上的時間開始閃爍。
  2. 遙控器：按 水量上 調整「小時」，按 水量下 調整「分鐘」。
    提示：每按一次微調一格，連續按可快速調整。
  3. 等待 5 秒 不操作，時間設定會自動儲存。
2-1

設定遙控器 ID（進階）

  1. 遙控器：長按 停止 → 時間閃爍。
  2. 遙控器：在時間閃爍時，再長按 水溫 → 進入 ID 模式（顯示 ID01…）。
  3. 遙控器：按 按摩 以切換不同 ID 值。
⚠️
若同場有多台座機或多支遙控器，請為每組配對設定 不同 ID，以避免互相干擾。
2-2

與馬桶本體配對

  1. 馬桶本體：按一下 停止鍵。
  2. 馬桶本體：長按 屁屁清洗鍵 → 電源燈閃爍（進入「準備接收」）。
  3. 遙控器：按 停止鍵 → 聽到 B 一聲，表示完成配對。
  4. 馬桶本體：再次按 停止鍵，將電源打開即可使用。
