Panasonic 空調排程完整操作教學

步驟 1 - 進入系統設定

從主畫面點選右上方 🔧 Settings，再點選 ⚙️ System Settings

步驟 2 - 新增排程群組名稱

點選 Schedule group name settings

輸入密碼 int2 並送出

選擇一筆群組 Select，按下 Sttng 編輯

可修改名稱 Name，右側的 Display 建議保留為 256 不變更

記得打勾 Valid 並按 Register

按下 ↩ 返回上一頁

步驟 3 - 指定室內機到群組

返回 ⚙️ System Settings → 點選 I/D unit settings

輸入密碼 int2 → 點選 Line No. → 選擇 Adaptor

再點選 Link → 選擇你需要設定的 LINK（例如：2-LINK1）

勾選室內機 Select，按下 Setng，選擇 ScheduleGroup → 指派排程群組(如果沒有顯示則該室內機為獨立控制)

後按下 Register 儲存

步驟 4 - 設定排程時間

回到主畫面 → 點選 📊 Oper./Status → 再點選 📅 Set schedule

選擇 Schedule setting → 勾選群組 → 點 Edit

設定開關時間、溫度、模式、風速等內容

可按 +New data 新增時段，完成後按 Register 儲存

步驟 5 - 套用年曆排程

從 📅 Set schedule → 點選 Calendar

你可以：

  • 手動點選日期 → 點選右側 Mode1 指定排程模式
  • 可以按住 Sche Pt1 2秒後可以重新命名
  • 或點下方 By day of the week → 選擇某週期範圍 → 套用週一至週日排程

完成後按 Register 儲存
